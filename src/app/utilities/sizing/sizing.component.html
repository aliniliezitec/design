Sizing

Notation

Sizing utilities that apply to all breakpoints, from xs to xl, have no breakpoint abbreviation in them. This is because those classes are applied from min-width: 0 and up, and thus are not bound by a media query. The remaining breakpoints, however, do include a breakpoint abbreviation.

The classes are named using the format nao-[property]-[size] for xs and [property]-[breakpoint]-[size] for sm, md, lg, and xl.

Where property is one of:

  • mw - for classes that set max-width.

Where size is one of:

  • 0 - for classes that set the max-width to unset.
  • 140 - (by default) for classes that set the max-width to 140px.
  • 150 - (by default) for classes that set the max-width to 150px.
  • 190 - (by default) for classes that set the max-width to 190px.
  • 200 - (by default) for classes that set the max-width to 200px.
  • 210 - (by default) for classes that set the max-width to 210px.
  • 220 - (by default) for classes that set the max-width to 220px.
  • 230 - (by default) for classes that set the max-width to 230px.
  • 290 - (by default) for classes that set the max-width to 290px.
  • 300 - (by default) for classes that set the max-width to 300px.
  • 310 - (by default) for classes that set the max-width to 310px.
  • 320 - (by default) for classes that set the max-width to 320px.
  • 380 - (by default) for classes that set the max-width to 380px.
  • 560 - (by default) for classes that set the max-width to 560px.
