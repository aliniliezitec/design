Nao System Icons

Install our custom icons font.

Place the nao-system-icon.css and the five font files in your font directory and include the css in your html.

Usage

There are two ways to include the icons in the html, using class or data-icon attribute:

Icon construction

All icons should be drawn on a 24x24px (3px stroke width for outlines), 16x16px (2px stroke width for outlines) or a 48x48px grid frame (6px stroke width for outlines). Each frame includes keylines and a one pixel padding on all sides as guidance.

Containers

In cases where the icon is contained inside a circle, the diameter of the circle can be determined by doubling the width of the icon. Icons should be optically centered within the container.

Circular container
Squared container

Size

The size of the icons is controlled by the font-size and for controlling the height you should use a line-height smaller then the font-size.

System Icons

Our icon system is inspired by big organization iconography. It aims to be bold, communicative, and hard working. Filled versions work well at small sizes in product while the outlined version works better in signage and marketing.
