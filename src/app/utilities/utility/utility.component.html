Utility classes

Utility classes are CSS class names that serve one particular purpose, and are names as such,

.nao-no-user-select

The .nao-no-user-select class makes the text of the element and its sub-elements not selectable.

Best used on html elements that have (click) event and are pressed multiple times rapidly.
