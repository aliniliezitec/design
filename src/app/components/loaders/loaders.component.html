Loader progress bar

API

Inputs Default Description
state Unset It's a NaoCrud.State, based on this it starts/stop the loading
progress 0 The current value for the progress bar. [0, max]
type 'primary' The color of the progress bar. 'primary'|'secondary'|'success'|'info'|'warning'|'danger'|'dark'
striped true If true, the progress bar will be displayed as striped.
animated true If true, the progress bar will be animated.
height 4px Height of the progress bar
showValue false If true, the current percentage will be shown in the xx% format.
