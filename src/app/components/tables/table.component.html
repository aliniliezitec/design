Simple table

Simple table with sticky header and scrollbar

  • You could modify table cells to have a min-width: <th scope="col" style="min-width: 200px">Name</th>
  • If you need a sticky header just add .nao-sticky-thead class to the <thead class="table-header nao-sticky-thead">
  • And if you need to have a fixed container with scrollbar just add height: 300px; and overflow: auto; to a custom class.
